## Part 2. MTV Pattern 적용해보기

### 1. MTV Pattern

- MVC 패턴을 기반으로 한 Model - View - Template 패턴
  - Model : View에서 주고 받는 데이터 형식을 정의, DataBase(DB)
  - View : 데이터를 처리하는 영역, MTV 중 핵심역할
  - Template : 사용자에게 보여지는 영역 (HTML, CSS, JS, 템플릿 언어 등이 이에 해당)
